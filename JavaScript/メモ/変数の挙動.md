# 変数の挙動
~~~
# 変数の挙動
let x = 1;
let y = x;
x = 5;
console.log(x); //5 
console.log(y); //1
~~~
簡単なデータであればyにはxの値がコピーされるので、更新しても最初の値のままになる
~~~
let x = [1.2]
let y = x;
x[0] = 10;　=>xの１の部分を10に変更
console.log(x); //[10,2]
console.log(y); //[10,2]
~~~
しかし配列やオブジェクトなど複雑なデータはコピーされずにyにはxの置き場所を指定されるのでこのようになる
もしコピーをしたければスプレット構文を使う
~~~
let x = [1.2]
let y = [...x];
x[0] = 10;
console.log(x); //[10,2]
console.log(y); //[1,2]
~~~
