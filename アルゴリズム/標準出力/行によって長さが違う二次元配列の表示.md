# 行によって長さが違う二次元配列の表示 
例)　自然数 N と N 個の要素の数列 M が与えられます。  
1 ≦ i ≦ N の各 i について、i 行目には以下の数列を出力してください。  
1 以上 M_i 以下のすべての自然数を昇順、半角スペース区切りで出力してください。

入力 4  
2 4 3 1  

出力  
1 2  
1 2 3 4  
1 2 3   
1
~~~
N = gets.to_i 
M = gets.split(' ').map(&:to_i)

(1..N).each do |i|
    (1..M[i - 1]).each do |j| 
        print "#{j}#{ j == M[i - 1] ? "\n" : " " }" 
    end 
end
