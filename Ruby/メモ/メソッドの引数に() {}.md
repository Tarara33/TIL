# メソッドの引数に(),{}をつけない
Rubyではメソッド呼び出し時に ()を省略できる。
~~~
# メソッド呼び出しの丸カッコは省略可能。
<%= stylesheet_link_tag("application", "data-turbo-track": "reload") %>

# 上は以下のように書いても同じ
<%= stylesheet_link_tag "application", "data-turbo-track": "reload" %>
~~~
***

また、***最後***の引数がハッシュの場合、波カッコは省略可能。
~~~
# 最後の引数がハッシュの場合、波カッコは省略可能。
<%= stylesheet_link_tag("application", { "data-turbo-track": "reload" }) %>

# 上は以下のように書いても同じ
<%= stylesheet_link_tag("application", "data-turbo-track": "reload") %>
~~~
***
