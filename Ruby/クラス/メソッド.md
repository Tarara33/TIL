# クラスメソッド
レシーバーがクラスのメソッド。(インスタンスは呼べない)
~~~

def class.メソッド名
end

または

class User
  class << self
    def メソッド名
    end
  end
end
~~~
の書き方二通りある。  
複数あるなら後者の方が、いちいちメソッド前に class.つけなくていいので楽。
***

## 例
~~~
class User
  def greet
    puts "HELLO"
  end

  def self.introduction
    puts "Userクラスです"
  end
end

user = User.new
user.greet
=> HELLO
user.introduction
=> undefined method `introduction' for #<User:0x000000014c868560> (NoMethodError)

User.introduction
=> Userクラスです
User.greet
=> undefined method `greet' for User:Class (NoMethodError)
~~~
💡 逆にインスタンスメソッドをクラスは使えない。
