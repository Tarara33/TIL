# uniq
配列から重複した要素を取り除いた新しい配列を返す。   
`uniq!` は削除を破壊的に行い、削除が行われた場合は `self`を、そうでなければ `nil`を返す。  
取り除かれた要素の部分は前に詰められる。  
~~~
p [1, 1, 1].uniq
=> [1]

p [1, 4, 1].uniq
=> [1, 4]

p [1, 3, 2, 2, 3].uniq
=> [1, 3, 2]
~~~
***

## ブロックの場合
ブロックが与えられた場合、ブロックが返した値が重複した要素を取り除いた配列を返す。
~~~
p [1, 3, 2, "2", "3"].uniq
=> [1, 3, 2, "2", "3"]

p [1, 3, 2, "2", "3"].uniq { |n| n.to_s }
=> [1, 3, 2]
~~~
