# to_s
文字列型へ変更する
~~~
123.to_s
=> "123"
~~~
***

# to_i
数値型へ変更する。
~~~
"123".to_i
=> 123
~~~
***

# to_a
ハッシュを配列型へ変更する。
~~~
hash = {a:1, b:2, c: 3}
hash.to_a
=> [[:a, 1], [:b, 2], [:c, 3]]
~~~
***

# to_h
配列をハッシュ型に変更する。
~~~
array = [[:a, 1], [:b, 2], [:c, 3]]
array.to_h
=> {:a=>1, :b=>2, :c=>3}
~~~
⚠️ ハッシュのキーと値に当てはまる配列なら変更できるが、当てはまらない場合([1,2,3,4]とか)エラーでる。
***

# to_sym
文字型をシンボル型に変更する。
~~~
'apple'.to_sym
=> :apple
~~~
***
