# 配列の結合
方法としては主に二つ
- 演算子を使う
- concatメソッドを使う

二つの違いは、    
演算子の場合、元の配列はどちらもそのままなのに対し、  
concatを使うと、レシーバーである配列は変更される(破壊される)
***

## 演算子で結合
[配列]　+ [配列]で結合する
~~~
a = [1,2,3]
b = [1,3,5]
c = ["a", "b", "c"]

a + b
=> [1, 2, 3, 1, 3, 5]
a
=> [1,2,3]
b
=> [1,3,5]

a + c
=> [1, 2, 3, "a", "b", "c"]
a
=> [1,2,3]
c
=> ["a", "b", "c"]
~~~
***

## concatメソッドを使う
[配列].concat([配列])
~~~
a = [1,2,3]
b = [1,3,5]
c = ["a", "b", "c"]

a.concat(b)
=> [1, 2, 3, 1, 3, 5]
a
=> => [1, 2, 3, 1, 3, 5]
b
=> [1,3,5]

a.concat(c)
=> [1, 2, 3, 1, 3, 5, "a", "b", "c"]
a
=> [1, 2, 3, 1, 3, 5, "a", "b", "c"]
c
=> ["a", "b", "c"]
~~~
