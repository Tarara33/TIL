# 和集合 (|)
配列の足し算。  
重複する要素はそのままである。
~~~
a = ["a", "A", 1, 2]
b = ["a", "b", 1, 3, 4]
a | b
=> ["a", "A", 1, 2, "b", 3, 4]
~~~
***

# 差集合 (-)
配列の引き算。  
左の配列から右の配列に含まれる要素を取り除く。
~~~
a = ["a", "A", 1, 2]
b = ["a", "b", 1, 3, 4]
a - b
=> ["A", 2]
~~~
***

# 積集合 (&)
二つの配列で共通する要素を出す。
~~~
a = ["a", "A", 1, 2]
b = ["a", "b", 1, 3, 4]
a & b
=> ["a", 1]
~~~
***

## 💡
いずれの集合も非破壊的なので元の配列は残る。
***
