# サポートモジュール
rspec内で何度も出てくる処理をまとめられる。    
例えば、ログイン処理とか。    
***

# ファイルの作り方
「spec/support/」配下に「〇〇_support.rb」というファイルを作る。(手動)    
〇〇はモジュール名(login_support.rbとか)
***

# ファイルにコードを書く
~~~
[spec/support/login_support.rb]

module LoginSupport
  def login(user)
    visit login_path
    fill_in 'Email', with: user.email
    fill_in 'Password', with: "password"
    click_button 'Login'
  end
end
~~~
