# Node.js
Node.jsのサイト(https://nodejs.org/ja) にとぶ。  
「ダウンロード」の項目を押してダウンロードページに行くと、LTS(安定版)のバージョンと    
npmのバージョンも書いてあるのでそれを参考にする。

## npm
npmは Node.jsとセットなので　Node.jsを新しくダウンロードしたら、npmもそれに対応したバージョンがダウンロードされる。
